⚙️ VSCodeのオススメ設定

設定項目内容操作手順
Auto Save onFocusChange に設定すると、ウィンドウ切り替え時に自動保存されます 左下の歯車（管理） →
[設定]を開く →
“Auto Save”を検索 →
“onFocusChange” を選択
Live Preview 拡張機能 リアルタイムでWebページをVSCode内でプレビュー可能 拡張機能タブを開く →
“Live Preview”を検索しインストール →
HTMLファイルで右クリック →
“プレビューの表示”を選択

🔤 HTMLの主なタグ一覧

タグ意味用途の例
<h1>〜<h6>見出しセクションのタイトル
<p>段落文章を囲む
<a href="">リンク他のページへ遷移
<img src="">画像画像を表示
<ul>/<ol>/<li>リスト箇条書きや番号付き
<div>/<span>汎用要素グルーピングや装飾
<head>メタ情報タイトルやCSSの読み込みなど
<body>本文表示されるすべての要素を含む
<nav>ナビゲーションページ内リンクをまとめる
<main>主要コンテンツそのページで最も重要な内容
<article>記事ブログ記事やニュース記事など独立した内容
<section>区切りトピックごとのグループ化
<aside>補足情報広告や関連リンクなど
<footer>ページの下部著作権や連絡先など
<table>表データの表形式表示
<tr>/<th>/<td>行/見出しセル/通常セル表の構成要素
<input>入力欄テキストやボタン、チェックボックスなど
<label>入力ラベルフォームでの説明文
